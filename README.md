# Ditto

Minecraftのダイアログ選択を記憶するシンプルで便利なMODです。

## 機能

- 「はい/いいえ」形式（`ConfirmScreen`）や「通知」形式（`NoticeScreen`）のダイアログに「次回から表示しない」チェックボックスを追加します。
- チェックを入れて選択すると、次回から同じダイアログが表示された際に、自動的に前回と同じボタンが押されます。
- バイパスキー（デフォルトは左Shift）を押しながらダイアログを開くことで、記憶された設定を無視して手動で選択することができます。
  - 設定画面から使用するキーを変更可能です。
- 日本語と英語をサポートしています。

## 使い方

1. MODを導入して起動します。
2. 対応しているダイアログが表示されると、下部に「次回から表示しない (Don't show again)」チェックボックスが表示されます。
3. チェックを入れて「はい」または「いいえ」をクリックすると、その選択が記憶されます。

### 設定の管理
Mod Menu が導入されている場合、ゲーム内の設定画面から以下の操作が可能です。
- **記憶された選択の管理**: 内容の変更（はい/いいえ の切り替え）、個別の削除、一括削除。
- **バイパスキーの設定**: 自動選択を一時的に無効化する際に使用するキーの変更。

## テスト・管理コマンド

動作確認や記憶のリセット用に、以下のコマンドを用意しています。

- `/ditto test 1`: ConfirmScreen (はい/いいえ) のテスト表示
- `/ditto test 2`: NoticeScreen (通知/OK) のテスト表示
- `/ditto clear`: 記憶されたすべての選択をリセット

## 導入推奨MOD

- **[Mod Menu](https://github.com/TerraformersMC/ModMenu)**: ゲーム内でMOD一覧を表示し、設定画面にアクセスするために必要です。
- **[Cloth Config API](https://github.com/shedaniel/cloth-config)**: このMODの設定画面を構築するために必要なライブラリです。

### 環境
- **Minecraft**: 1.21.11
- **Fabric Loader**: 0.18.0 以上
- **Java**: 21

