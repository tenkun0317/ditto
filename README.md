# Ditto

Minecraftのダイアログ選択を記憶するシンプルで便利なMODです。

## 機能

- 「はい/いいえ」形式のダイアログ（`ConfirmScreen`）に「次回から表示しない」チェックボックスを追加します。
- チェックを入れて選択すると、次回から同じダイアログが表示された際に、自動的に前回と同じボタンが押されます。
- 日本語と英語をサポートしています。

## 使い方

1. MODを導入して起動します。
2. 対応しているダイアログが表示されると、下部に「次回から表示しない (Don't show again)」チェックボックスが表示されます。
3. チェックを入れて「はい」または「いいえ」をクリックすると、その選択が記憶されます。

### 設定の解除方法
記憶された選択をリセットしたい場合は、Minecraft の `config` フォルダ内にある `ditto.json` を削除、またはエディタで編集してください。

## テスト用コマンド

動作確認用に、メッセージの長さが異なる3種類のダイアログを表示するコマンドを用意しています。

- `/dittotest 1`: 短いメッセージ（5行）
- `/dittotest 2`: 中くらいのメッセージ（10行）
- `/dittotest 3`: 長いメッセージ（15行）

### 環境
- **Minecraft**: 1.21.11
- **Fabric Loader**: 0.18.0 以上
- **Java**: 21

### ビルド
```bash
./gradlew build
```
